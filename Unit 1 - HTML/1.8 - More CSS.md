## Self-Discovery of CSS and HTML

🚀 Here's a "Level Up" Discovery Checklist

Create some more content / pages and learn through doing!

### Layout & Spacing (The "Architecture")
☐ **The Box Model.** Learning [the box model](https://www.w3schools.com/css/css_boxmodel.asp) is crucial to understanding CSS. Experiment with padding (space inside the border) vs. margin (space outside the border).

☐ **The Container ([`<div>`](https://www.w3schools.com/tags/tag_div.asp)).** Potentially the most powerful (or at least most used) HTML tag of them all. Wrap a few paragraphs and an image in a `<div>`. Give that div a background-color and padding to create a "content card."

☐ **The Divider (`<hr>`).** Drop a `<hr>` tag in your HTML to create a clean horizontal line between sections.

☐ **Border Basics.** Use border: 2px solid black; to put a frame around any element (try a `div` or `<p>`)

> Now's a good time to make a commit and push it to Gitea!

### Visual Style (The "Paint")
☐ **Rounded Corners.** Use [border-radius](https://www.w3schools.com/cssref/css3_pr_border-radius.php): 10px; on images or boxes. (Try 50% to make a square image a circle!)

☐ **Shadows.** Use [box-shadow](https://www.w3schools.com/cssref/css3_pr_box-shadow.php): 5px 5px 10px gray; to make an element "pop" off the page.

☐ **Transparency.** Use [opacity](https://www.w3schools.com/css/css_image_transparency.asp): 0.5; to make an image or section see-through.

☐ **Link Cleanup.** Use [text-decoration](https://www.w3schools.com/cssref/pr_text_text-decoration.php): none; on an `<a>` tag to remove the default underline.

☐ **The "Button Maker"**. Turn that plain link you just made into a button! Use text-decoration: none; but add a background color and some padding. Voila! A button!

☐ **Center Everything.** Use [text-align](https://www.w3schools.com/cssref/pr_text_text-align.php): center; on a parent element to align all the text and images inside it.

> Now's a good time to make a commit and push it to Gitea!

### Interaction (The "Magic")
☐ **Hover States.** Use the `:hover` selector to change a color (heading, paragraph, etc) only when the mouse cursor moves over it.

Example: a:hover { color: red; }

☐ **Smooth Transitions.** Add transition: 0.5s; to an element so that when the color changes on hover, it _fades_ instead of snapping.

☐ **The "Accordion" (HTML)** Use the `<details>` and `<summary>` tags to create a clickable dropdown without any code.

> Now's a good time to make a commit and push it to Gitea!
